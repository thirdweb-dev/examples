# thirdweb Gasless NFT drop

## Description

This project shows you how to make a gasless NFT drop using the Edition Drop Contract of the thirdweb Javascript sdk.

### Technologies Used

- Thirdweb
- Next.js
- Openzeppelin

### thirdweb SDK Functionality Used

- editionDrop.claimTo

### Running the project

- `yarn install`
- `yarn dev`
